# textures

Place texture and thumbnail files here. Required files before running the application:

| File | Used by |
|------|---------|
| `cream-leather.jpg` | Cream Leather material preset (albedo map) |
| `cream-leather-thumb.jpg` | Cream Leather material preset (thumbnail) |
| `blue-velvet.jpg` | Blue Velvet material preset (albedo map) |
| `blue-velvet-thumb.jpg` | Blue Velvet material preset (thumbnail) |
| `walnut-wood.jpg` | Walnut Wood material preset (albedo map) |
| `walnut-wood-thumb.jpg` | Walnut Wood material preset (thumbnail) |

Files are referenced by `src/data/materialPresets.ts`.
